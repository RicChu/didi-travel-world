## 目前行程計劃進度

目前大方向已成立：城市順序、主要跨城交通、11/02 格雷梅日出熱氣球、以弗所 / 棉堡自駕、卡帕多奇亞兩段住宿、伊斯坦堡 Basilissis Hotel 四晚基地、兩段租車、兩段 Klook 機場接送與 Klook Pass 土耳其都已排進主線。接下來的重點不是重新設計路線，而是補齊旅平險、租車執行細節、離線備份、Klook Pass 活動預約與餐廳低消 / 訂位確認，讓現場可以直接照表執行。

### 完整度判斷方式

這份計劃後續要做到「時 / 分」顆粒度，但不代表每個景點都要精準到分鐘。建議用下面標準判斷完整度：

| 層級 | 完整度定義 | 目前狀態 |
|------|------------|----------|
| 路線骨架 | 城市順序、住宿城市、跨城方向已確定 | 約 98%，跨城方向已鎖定為伊斯坦堡抵達 → 以弗所 / 棉堡 → 卡帕多奇亞 → 伊斯坦堡收尾；伊斯坦堡四晚基地改為 Basilissis Hotel，舊城、新城、亞洲側與回機場動線已有固定起訖點 |
| 預定錨點 | 航班、住宿、租車、熱氣球等不可隨意漂移的項目 | 約 98%，國際線、TK2320 IST → ADB、ADB → ASR、TK2017 ASR → IST、以弗所住宿、卡帕多奇亞住宿、Basilissis Hotel 四晚、兩段租車、11/02 熱氣球、11/04 接機、11/08 送機與 Klook Pass 土耳其已完成；尚待旅平險 |
| 時分級動線 | 每天有起訖時間、移動時間、景點停留、用餐與行李安排 | 約 85%，10/31～11/04 已可按時段執行，紅線 / 綠線與熱氣球備援邏輯已成形；11/04 晚～11/08 已以 Basilissis Hotel 為基地補入機場接送、步行、T1 電車、渡輪與叫車策略 |
| 現場可執行度 | 有 Plan B、取消條款、聯絡方式、地址、截圖與離線備份 | 約 86%，熱氣球票券、天候退款規則、土航國內線電子機票、行李額度、租車確認信、11/04 泳衣拍照備註、伊斯坦堡住宿、Klook 機場接送與 Klook Pass 預訂已補；尚需補 Klook Pass 活動時段、旅平險、土耳其 e-Visa、AVEC 提早還車 / 保險細項、Basilissis Hotel 取消條款 / 總價 / 是否含早餐、離線地圖與出發前一週救急版 |

### 下一步優先順序

1. 先處理會影響後段體驗的剩餘項目：Klook Pass 於 2026/08/12 開放後預約活動時段、Basilissis Hotel 取消條款 / 總價 / 是否含早餐補齊、旅平險。
2. 再確認已訂項目的執行細節：兩段 Rentalcars / AVEC 租車保險與滿油規則、取車信用卡、11/01 Kayseri 深夜取車、11/04 因 TK2017 18:40 起飛需提早還車、熱氣球實際接送時間與 11/03 / 11/04 改期座位。
3. 最後細化伊斯坦堡：Seven Hills 少量點餐 / 低消規則、Olden 1772 是否訂位、11/08 午餐店名與夜間散步範圍。

時間顆粒度建議：跨城日抓到 15～30 分鐘區間；景點日抓到 30～60 分鐘區間；伊斯坦堡生活街區和晚餐保留 1～2 小時彈性，不需要每個小店都鎖死。
