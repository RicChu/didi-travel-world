## 目前行程計劃進度

目前大方向已成立：城市順序、主要跨城交通、11/02 格雷梅日出熱氣球、以弗所 / 棉堡自駕、卡帕多奇亞兩段住宿、伊斯坦堡 Basilissis Hotel 四晚基地、兩段租車、兩段 Klook 機場接送與 Klook Pass 土耳其都已排進主線。接下來的重點不是重新設計路線，而是補齊旅平險、租車執行細節、離線備份、Klook Pass 活動預約與餐廳低消 / 訂位確認，讓現場可以直接照表執行。

## 卡帕多奇亞紅線 / 綠線疲勞度

結論：**綠線比紅線累**。綠線累在距離長、點位分散、地下城與 Ihlara / Selime 會消耗體力；紅線累在景點密度高、拍照點多，但大多集中在 Göreme / Avanos 周邊，車程短很多。

| 項目 | Green 線 | Red 線 |
|------|----------|--------|
| 疲勞度 | 高，尤其熱氣球後再跑完整 Ihlara / Selime | 中，景點多但移動短 |
| 自駕時間 | 完整版約 4～5 小時；精簡版約 2～2.5 小時 | 約 2～3 小時，含回 Grand Elite 拿行李與開去 Gamirasu |
| 景點時間 | 完整版約 4～5 小時；精簡版約 2～3 小時 | 約 4～5 小時 |
| 一天總長度 | 完整版約 8～10 小時；精簡版約 4～6 小時 | 約 6～8 小時 |
| 最累原因 | 長距離自駕 + 地下城 + Ihlara Valley / Selime | 景點密集 + 退房寄放行李 + 換飯店 |
| 建議用法 | 適合放在沒搭到熱氣球、體力較完整的那天；若前一天已完成 red 線，隔天再走 green 線 | 適合放在熱氣球飛成的那天，或 11/03 需要銜接換飯店時；若接駱駝，red 線必須精簡 |

紅線 / 綠線 / 駱駝決策規則：**熱氣球飛成的那一天，白天排 red 線但不騎駱駝；熱氣球沒飛成且體力完整的第一天，白天排 green 線；駱駝只排在非熱氣球日，優先 11/04 11:00，若 11/04 可能變成熱氣球最後備援日，才改到 11/03 14:00 接 red 線精簡版。** 這樣可避免清晨熱氣球後再硬跑最累的 green 線或再加駱駝。

### 夜間火車判斷

安安與豬豬可接受夜間火車，但本行程先把它列為**備選**，不是優先。原因：

- 土耳其境內長距離火車路線不一定剛好連接這次的必去點。
- 有臥鋪不代表一定有可接受的淋浴設備，需以訂票頁或車廂資訊確認。
- 帶 26 / 28 吋大行李箱時，半夜上下車、轉乘、找月台會比較累。
- 若夜間火車無法明確確認行李位置與淋浴條件，建議改搭飛機或包車。

## Google Maps 使用建議

1. 在 Google Maps 建一個清單，例如：`2026 Turkey`
2. 先收藏機場、住宿、每天想去的景點與交通點
3. 用標籤區分：
   - `必去`
   - `想去`
   - `咖啡`
   - `雨天備案`
   - `行李寄放`
4. 每天出門前只看當天同區域的點，避免行程太散

如果想做更完整的地圖路線，可以再建立 [Google My Maps](https://www.google.com/maps/d/)，把每天用不同顏色分層。
