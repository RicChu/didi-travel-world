#### 10/31（六）伊斯坦堡機場 → Izmir → 以弗所周邊

疲勞度：**高**。夜航抵達後接非同張票國內線、入境領行李、重新托運、取車與下午以弗所都在同一天；若航班或取車延誤，直接啟動以弗所精簡版。

景點背景速覽：

- 以弗所是愛琴海沿岸最重要的古城之一，曾是羅馬帝國亞洲行省的重要城市，重點不是單一建築，而是沿著古代城市主街看公共生活、宗教、商業與劇場空間如何串在一起。
- 塞爾蘇斯圖書館是以弗所最具代表性的門面建築，原本兼具圖書館與紀念墓功能；大劇院則呈現古城公共娛樂與集會規模，適合用來感受城市曾經的繁華。
- Selçuk 是以弗所旁的生活型小鎮，晚上住這裡比住海邊度假區更順路，重點是降低隔天去棉堡的交通壓力。

餐食建議：

- 午餐：在伊斯坦堡機場候機時隨便吃（與早餐合併在 IST 解決），下午不繞 Selçuk 鎮中心午餐。
- 晚餐：已訂 Selçuk 的 Pikan Ephesus，隔天自駕去棉堡比住 Kusadasi 單純；晚上選鎮中心附近即可。
- 刪減規則：11:00 國內線若延誤或取車拖到 13:30 後，以弗所只保留 Library of Celsus、The Great Theatre、Curetes Street。

#### 11/01（日）以弗所周邊 → 棉堡 → 卡帕多奇亞

疲勞度：**很高**。這天是全行程最硬交通日，自駕距離長、晚上還要還車、搭機、深夜取車與開夜路進 Göreme；重點是保留安全與準時。**Cleopatra Antique Pool 泡澡已列入必排**，出門前準備泳衣。

景點背景速覽：

- 棉堡的白色石灰棚來自含鈣溫泉水長期沉積，視覺重點是水、礦物與坡地形成的地貌，不是人工建築；光線好時層次最明顯。
- Hierapolis 是建在溫泉旁的古城，古人把這裡視為療養與宗教場所，因此棉堡和古城要一起看，才會理解為什麼人們會在這片地形旁建立城市。
- Cleopatra Antique Pool（古董溫泉池）的賣點是泡在古代羅馬遺構與溫泉之中。

餐食與節奏：

- 午餐：Pamukkale 村可解決，但偏觀光；以準時回 ADB 為優先。
- 晚餐：建議在 ADB 機場簡單吃，抵達卡帕多奇亞已經很晚。
- 刪減規則：若時間壓力變大，**不刪古董池泡澡**；改壓縮石灰棚拍照時間或略過 Hierapolis 古城其他區域，只保留劇場。

#### 11/05（四）伊斯坦堡收尾第 1 天

疲勞度：**低中**。前一晚已回到伊斯坦堡並入住 Basilissis Hotel，這天只做城市暖身與短距離散步；白天以步行、T1 電車和短程叫車銜接，不把跨區移動排太滿。

今日目標：補齊必去清單中的「土耳其式早餐」、「伊斯坦堡城市體驗」與 Karakoy / Beyoğlu 街頭美食。Basilissis Hotel 位於 Alemdar / Gülhane 附近，這天作為城市暖身日，早餐固定排 Boris'in Yeri，先完成比較漂亮好拍的傳統早餐，再轉往 Galata Bridge、Karakoy / Galata 周邊，下午插入 Balık Dürüm Mehmet Usta，晚上視胃口安排 Suat Usta Tantuni。

景點背景速覽：

- Karakoy / Galata 是從港口、商業、外國商館到現代咖啡店與餐廳混合的區域，適合當作回到伊斯坦堡後的低強度城市暖身。
- Boris'in Yeri 是 Kumkapı / Fatih 一帶的老早餐店，主打傳統土耳其早餐與乳製品；比原本的新城早餐點更靠近 Basilissis Hotel，也更適合作為好拍的早餐場景。
- Sokak Lezzeti Tarihi Balık Dürümcü Mehmet Usta 在 Karakoy，適合當作魚捲點心 / 輕午餐，不要和 Boris'in Yeri 都吃到太滿。
- Suat Usta Mersin Tantuni 比較適合放晚上或另一晚當輕鬆晚餐；若 11/05 胃口不夠，直接順延，不犧牲白天動線。
- Galata Bridge 的看點是城市日常，橋上釣魚、橋下餐廳、金角灣與舊城天際線會同時出現，適合觀察伊斯坦堡作為港口城市的節奏。

#### 11/06（五）伊斯坦堡舊城區

疲勞度：**中**。舊城景點密集、排隊與步行會累，但不用換城市；重點是控制排隊時間，不要把大景點和兩個市集全塞滿。

今日目標：補齊必去清單中的「伊斯坦堡大景點」主體，把舊城核心集中處理：藍色清真寺、聖索菲亞、地下水宮殿，中午用 Seven Hills Restaurant 看屋頂景觀與拍照，但只少量點餐；下午固定去 Grand Bazaar 大巴札，並把 Hafiz Mustafa 1864 Sirkeci 放在這天當甜點茶點。Topkapi Palace 移到 11/08 一早走短版。

景點背景速覽：

- Sultanahmet 舊城是拜占庭君士坦丁堡與鄂圖曼帝國核心權力區重疊的地方，同一片區域內可以看到教堂、清真寺、地下水利與市集。
- Hagia Sophia 原是拜占庭帝國的大教堂，後來成為鄂圖曼清真寺，再經歷博物館與清真寺身分變化；它的重點是巨大的穹頂空間與不同宗教層次共存。
- Blue Mosque 是鄂圖曼時期的重要清真寺，六座宣禮塔與藍色磁磚是特色；Basilica Cistern 則是古代地下蓄水系統，和地面上的宗教建築形成很強對比。
- Grand Bazaar / Spice Bazaar 都是市集，但大巴札偏工藝、金飾與觀光商品，香料市集偏食物與香氣；11/06 固定去 Grand Bazaar，11/08 不再補市集。
- Seven Hills Restaurant 重點是 Sultanahmet 屋頂視角，適合把聖索菲亞、藍色清真寺和海景一次收進拍照體驗；因為價位高，不當正式午餐，優先點飲料、甜點、湯品或小菜。
- Hafiz Mustafa 1864 Sirkeci 適合做舊城日甜點茶點，不適合放在 11/08 一早當正餐；兩人分食 1～2 份甜點即可。
- Suat Usta Mersin Tantuni 不在 Sultanahmet / Sirkeci / Fatih 附近，最近主線仍偏 Taksim / Beyoğlu；不建議 Seven Hills 後特地繞去吃。

#### 11/07（六）伊斯坦堡儀式感 + 日常觀察

疲勞度：**低中 / 中**。早餐與土耳其浴偏放鬆，真正的疲勞來自下午是否跨去 Kadikoy / Uskudar；生活街區擇一即可。

今日目標：補齊必去清單中的「土耳其浴」，下午用 Kadikoy / Uskudar 或 Karakoy / Galata 擇一補日常觀察；11/05 已完成土耳其式早餐，這天早餐改輕量，避免土耳其浴前吃太重。傍晚先完成超市伴手禮，晚上以 Olden 1772 作近距離晚餐候選。

景點背景速覽：

- 11/05 已用 Boris'in Yeri 完成土耳其式早餐，今天早餐只保留輕量版本，讓土耳其浴和下午生活街區更舒服。
- Hammam 土耳其浴源自公共浴場文化，結合清潔、放鬆與儀式感；觀光型 hammam 比較容易預約與理解流程，傳統浴場則更在地但規則感更強。
- [Vefa Bozacisi](https://maps.app.goo.gl/scQBEiTX13GjXswQA) 是 1876 年開業的 Vefa 老字號，主打 **boza**（發酵小米飲，常配烤鷹嘴豆）；浴後順路來一杯當收尾，比正式午餐輕，也符合土耳其老派習慣。

- Galata / Karakoy 適合夜間短散步，能把港口、橋、塔與餐飲街區串起來，是低風險補夜景的選擇。
- Olden 1772 在 Sirkeci / Fatih 一帶，離 Basilissis Hotel 近，適合超市採買後先回飯店放東西，再出門吃一頓比較有儀式感的晚餐。
