## 卡帕多奇亞紅線 / 綠線疲勞度

結論：**綠線比紅線累**。綠線累在距離長、點位分散、地下城與 Ihlara / Selime 會消耗體力；紅線累在景點密度高、拍照點多，但大多集中在 Göreme / Avanos 周邊，車程短很多。

| 項目 | Green 線 | Red 線 |
|------|----------|--------|
| 疲勞度 | 高，尤其熱氣球後再跑完整 Ihlara / Selime | 中，景點多但移動短 |
| 自駕時間 | 完整版約 4～5 小時；精簡版約 2～2.5 小時 | 約 2～3 小時，含回 Grand Elite 拿行李與開去 Gamirasu |
| 景點時間 | 完整版約 4～5 小時；精簡版約 2～3 小時 | 約 4～5 小時 |
| 一天總長度 | 完整版約 8～10 小時；精簡版約 4～6 小時 | 約 6～8 小時 |
| 最累原因 | 長距離自駕 + 地下城 + Ihlara Valley / Selime | 景點密集 + 退房寄放行李 + 換飯店 |
| 建議用法 | 適合放在沒搭到熱氣球、體力較完整的那天；若前一天已完成 red 線，隔天再走 green 線 | 適合放在熱氣球飛成的那天，或 11/03 需要銜接換飯店時；若接駱駝，red 線必須精簡 |

紅線 / 綠線 / 駱駝決策規則（見 [行程.md](./行程.md) Plan A/B/C）：**以 Plan A 為預設**（11/02 熱氣球飛成 → 當天精簡紅線 + **15:00 駱駝**；駱駝山谷與 Ortahisar 自駕跳過）。熱氣球未飛的第一個完整白天走 **綠線精簡**；紅線與駱駝排在熱氣球飛成日或 Plan C 的 11/03。**Avanos 陶藝固定 11/04 下午**。

### 每日疲勞度總覽

疲勞度以交通時間、早起程度、換飯店 / 取還車、景點步行量與是否有深夜抵達綜合判斷。等級定義：`低` = 可慢走與休息；`中` = 正常景點日；`中高` = 需要控體力；`高` = 當天容易累積疲勞；`很高` = 需要預留刪減方案。

| 日期 | 疲勞度 | 判斷原因 | 優先刪減 / 保護規則 |
|------|--------|----------|----------------------|
| 10/30（五） | 高 | 長途夜航出發，睡眠品質不確定 | 不安排抵達前額外任務，護照、行李、轉機文件優先 |
| 10/31（六） | 高 | 國際線抵達、入境領行李、非同張票轉國內線、取車與以弗所下午行程接在同一天 | 若 TK2320 或取車延誤，以弗所只保留圖書館、大劇院、Curetes Street |
| 11/01（日） | 很高 | 全行程最硬交通日，自駕棉堡來回、ADB 還車、晚班機、ASR 深夜取車與夜路進 Göreme | **Cleopatra Antique Pool 泡澡為必排**（需泳衣）；若時間壓力大，壓縮石灰棚拍照或 Hierapolis 古城散步，不刪泡池；晚上只入住睡覺 |
| 11/02（一） | 中高 / 高 | Plan A：熱氣球 + 精簡紅線 + 15:00 駱駝；Plan B/C：綠線精簡 | 飛成後紅線刪 Devrent / Love Valley；取消則綠線只留地下城一個 + Uçhisar 或鴿子谷 |
| 11/03（二） | 中高 / 高 | Plan A：綠線 + 換 Gamirasu；Plan B：熱氣球 + 紅線 + 駱駝 + 搬家（最硬）；Plan C：紅線 + 駱駝 + 搬家 | Plan B 紅線只留 Museum + Pasabag；綠線刪 Ihlara / Selime |
| 11/04（三） | 中 / 中高 | Plan A/B：自然醒 + Avanos 陶藝 + ASR；Plan C 補飛：熱氣球 + 趕航班 | Plan C 補飛日陶藝只能短版；三天都沒飛則不騎駱駝、申請退款 |
