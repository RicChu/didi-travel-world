# 花東四日行程計劃

## 7/15 三｜桃園 → 宜蘭 → 花蓮
住花蓮

| 時間 | 行程 |
|------|------|
| 08:00 | 桃園出發 |
| 10:00 | 宜蘭休息、午餐 |
| 12:30 | 往花蓮 |
| 15:30 | 抵達花蓮 |
| 16:00 | STARBUCKS 星巴克 花蓮理想門市 |
| 17:00 | 雲山水夢幻湖 |
| 18:30 | 花蓮公正包子 |
| 19:30 | 東大門夜市 |

---

## 7/16 四｜花蓮 → 池上 → 鹿野 / 台東
住台東市區或鹿野

| 時間 | 行程 |
|------|------|
| 09:00 | 花蓮出發 |
| 11:30 | 池上午餐 |
| 12:30 | 金城武樹 |
| 14:30 | 初鹿牧場 |
| 17:00 | 鹿野高台 |
| 晚上 | 住台東 |

---

## 7/17 五｜台東 → 太麻里 → 花蓮
住花蓮

| 時間 | 行程 |
|------|------|
| 09:00 | 台東出發 |
| 10:00 | 太麻里櫻木平交道 |
| 11:30 | 台東或太麻里午餐 |
| 13:30 | 往花蓮 |
| 17:30 | 抵達花蓮 |
| 晚上 | 可輕鬆補逛東大門夜市，或休息 |

---

## 7/18 六｜花蓮 → 宜蘭 → 土城

| 時間 | 行程 |
|------|------|
| 08:30 | 花蓮出發 |
| 11:30 | 抵達宜蘭 |
| 11:30–12:30 | 宜蘭午餐 |
| 12:30 | 弈順軒買生乳捲（建議礁溪店或宜蘭店，看當天路線與塞車狀況）|
| 13:30 | 出發回土城 |
| 15:00–16:30 | 抵達土城 |
