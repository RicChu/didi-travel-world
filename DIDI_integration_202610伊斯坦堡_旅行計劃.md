## 卡帕多奇亞紅線 / 綠線疲勞度

結論：**綠線比紅線累**。綠線累在距離長、點位分散、地下城與 Ihlara / Selime 會消耗體力；紅線累在景點密度高、拍照點多，但大多集中在 Göreme / Avanos 周邊，車程短很多。

| 項目 | Green 線 | Red 線 |
|------|----------|--------|
| 疲勞度 | 高，尤其熱氣球後再跑完整 Ihlara / Selime | 中，景點多但移動短 |
| 自駕時間 | 完整版約 4～5 小時；精簡版約 2～2.5 小時 | 約 2～3 小時，含回 Grand Elite 拿行李與開去 Gamirasu |
| 景點時間 | 完整版約 4～5 小時；精簡版約 2～3 小時 | 約 4～5 小時 |

### 夜間火車判斷

安安與豬豬可接受夜間火車，但本行程先把它列為**備選**，不是優先。原因：

- 土耳其境內長距離火車路線不一定剛好連接這次的必去點。
- 有臥鋪不代表一定有可接受的淋浴設備，需以訂票頁或車廂資訊確認。
- 帶 26 / 28 吋大行李箱時，半夜上下車、轉乘、找月台會比較累。
- 若夜間火車無法明確確認行李位置與淋浴條件，建議改搭飛機或包車。

### 租車 / 包車放行李的風險

租車或包車把行李放車上通常比拖著行李走景點方便，但不是零風險。10/31～11/01 若採 ADB 機場租車自駕，行李需要全程用後車廂或遮簾遮住，貴重物品永遠隨身。

| 風險 | 說明 | 降低方式 |
|------|------|----------|
| 司機偷竊 | 正規公司或飯店安排的司機風險較低，但仍不應完全信任 | 用飯店、Klook / KKday / GetYourGuide 高評價供應商，保留車牌與司機資訊 |
| 車窗被砸 / 車被闖入 | 景點停車場若行李可被看見，風險上升 | 行李放後車廂，不外露；護照、錢、電腦、相機隨身 |
| 轉包司機 | 供應商臨時換車換司機時，責任界線可能不清 | 出發前確認公司名稱、車牌、司機姓名、聯絡方式 |
| 保險責任不明 | 行李遺失不一定由司機或平台賠償 | 旅平險確認行李遺失 / 竊盜理賠條件 |
| 行李尺寸不合 | 26 / 28 吋兩件大行李可能塞不進小車 | 租車時確認車型、後車廂容量與遮物簾，必要時升級車型 |

自駕停車時補充規則：

- 優先租轎車，需有封閉後車廂，並確認可放 1 個 26 吋 + 1 個 28 吋行李箱。
- 不要在以弗所、棉堡、休息站停車場打開後車廂整理行李，避免讓旁人看到車內有大箱。
- 停車後車內不要留下背包、外套、購物袋、電子用品線材或任何看起來可拿的東西。
- 以弗所與棉堡都優先停正式收費停車場或有管理員的停車場。
- 11/01 還車時間可保留到 20:00 避免租期超時，但實際建議 18:30～19:00 抵達 ADB 還車，保留加油、驗車、托運與安檢時間。

### 貴重物品規則

- 護照、信用卡、現金、手機、相機、電腦、藥品：**永遠隨身**。
- 租車或包車時，行李箱可放車上，但貴重物品不要放行李箱。
- 如果要去熱氣球、棉堡、以弗所這種戶外景點，建議每天另準備一個小背包。

### 卡帕多奇亞三晚與熱氣球備案

卡帕多奇亞三晚是為了保留三個熱氣球清晨，同時不壓縮後段伊斯坦堡。熱氣球很看風速與天候，若某天清晨取消，通常不是單一公司取消，而是當地航空主管機關不允許當天飛行。這版 11/01 晚上抵達卡帕多奇亞並入住洞穴飯店，11/02、11/03、11/04 三個清晨可嘗試熱氣球；11/04 晚上若有合適班機可回伊斯坦堡，否則 11/05 白天再飛。

若第一天就順利搭到熱氣球，剩下的卡帕多奇亞時間不會浪費，可以改成更慢、更有空間感的地面體驗：

- 天氣只是風大但不下雨：改走 Red Tour 或包車，安排 Göreme Open Air Museum、Pasabag / Monks Valley、Devrent Valley、Avanos 陶藝小鎮。
- 想維持空間感與夕陽體驗：白天走 Uçhisar Castle、Pigeon Valley、Ortahisar Castle，傍晚視天氣安排 Göreme Sunset View Point、ATV sunset tour 或短程觀景散步。
- 若下雨或很冷：安排 Kaymakli / Derinkuyu 地下城、Avanos 陶藝體驗、地毯店 / 咖啡館、洞穴飯店 spa / 土耳其浴，晚上早點休息，準備隔天補飛。

## Google Maps 使用建議

1. 在 Google Maps 建一個清單，例如：`2026 Turkey`
2. 先收藏機場、住宿、每天想去的景點與交通點
3. 用標籤區分：
   - `必去`
   - `想去`
   - `咖啡`
   - `雨天備案`
   - `行李寄放`
4. 每天出門前只看當天同區域的點，避免行程太散

如果想做更完整的地圖路線，可以再建立 [Google My Maps](https://www.google.com/maps/d/)，把每天用不同顏色分層。
